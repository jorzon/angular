Pasos para usbir el proyecot a github

  1. git init: al ingresar este comando, generamos una fotografia de nuestro proyecto
  2. git add -A: agregamos todos nuestros cambios
  3. git commit -m "resumen" : comentamos nuestros cambios
  4. git remote add origin https://github.com/jorzon/angular.git : con este comando indicamso qye vamos a agregar codigo a un repositorio remoto
  5. git branch -M main: le decimos a git que vamos a renombrar nuesrtra rama por "main"
  6. git push -u origin main: solo ingresamos este comando la primera vez.
de esa manera subimos nuestros proyecto a nuestro repositorio remoto. Para futuros cambios solo utilizamos los comando número 2,3 y para subir los cambios solo usamos el "git push"
git checkout -- . : este comando lo utilizamos para deshacer todo los cambios que han habido desde que tomamos la fotografia
SI CLONAMOS NUESTRO REPOSITORIO EN OTRA COMPU, TENEMOS QUE CLONAR EL REPOSITORIO Y LUEGO INGRESAR EL COMANDO npm install, que nos ayudara a instalar toda las dependencias
