INTRO SOBRE ANGULAR

trabaja sobre clases

  • angular.json : es para hacer configuraciones
  • componentes: cuenta con 3 partes, html, css y un TS (pero se podria solo usar un TS, y el html y el css puede ser en linea) nuestros componentes estan encapsulados
  • servicios
  • modulos
  • directivas principales: ngIf - ngFor
    *ngIf: nos ayuda a ocultar (destruir, no existe) elementos HTML
    *ngFor: nos ayuda a duplicar elementos html
  • archivos .spec: sirven para el testing

¿Que podemos hacer con angular?

Podemos desarrollar para un monton de plataformas: aplicaciones de windonw-mac (electron), aplicaciones mobile (ionic aplicaciones hibridas), ssr (angular universal)

¿Que es angular?

Es un marco de trabajo estandarizado, viene con todo lo que necesitas para trabajar, es modular, tiene el soporte de GOOGLE


BLOQUES FUNDAMENTALES DE ANGULAR

  • componentes: bloque de codigo HTML y CSS con su TS, se busca que sean bloques pequeños
  • rutas: mostrar diferentes componentes basados en la URL
  • directivas: existe 3 tipos:
    • directivas de componentes: ayuda a reutilizar codigo html con cierta funcionalidad
    • directivas estructurales: modifica el DOM
    • directivas de atributo: cambian el comportamiento o la apariencia de un elemento
  • servicios: lugares para centralizar la informacion de tu aplicacion - singleton
  • modulos: permite encapsular nuestra información
