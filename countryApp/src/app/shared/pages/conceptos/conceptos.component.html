Conceptos

Solo usamos unicamente un forRoot en toda nuestra aplicación, para el resto se usa forChild


Creamos un componente que viene con las rutas router-outlet, este se encarga de mostrar el contenido según la URL


Cuando trabajamos de manera encapsulada, debemos importar RouterModule de angular/router


Para activar el boton según la URL, debemos agregar la porpiedad routerLinkActive y la clase que qeremos ponerle


Cuando colocamos el path vacio, para evitar problemas al controlar las rutas activas agregamos [routerLinkActiveOptions]="exact: true", para decirle que el path debe ser exactamente vacio. va entre llaves


Cada ruta puede tener un router interno
