Señales - Signal

Las señales son una forma de llegar a una variable de forma directa y actualzarla o renderizarla.
tres formas de actualizar, modificar una señal set(establecer) , update(actualizar) y newstate(mutar)

Las señales se limpian de manera automatica cuando ya no se usan

Las FUNCIONES COMPUTADAS son para menajr datos de solo lectura pero tambien para usarse para calcular otros valores que ya se manejan.

En las versions de angular +17, ya no se usa el mutate, el update recibio mejoras

keyof User lo usamos para recibir como parametros solamente keys de nuestra interfaz, para evitar errores de tipeo

Los efectos en las señales son para disparar algun efecto o acción ante un cambio de nuestra señal o señales. LO IMPORTANTE ES QUE TIENE LIMPIEZA AUTOMATICA
